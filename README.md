# DevOps_Assignment

- Named the Project DRAGOT

---

## Objectives

- Create an end-to-end DevOps setup using Azure DevOps by creating a project, repository, and pipeline to build a sample Java/Python application, GITHUB as repo.
- Provision Azure Resources including an Azure Subscription, resource Group, and Azure Kubernetes Services (AKS) cluster/WebApp. 
-	Containerize the application using Docker and push the image to Azure Container Registry (ACR). 
- Create the Pipeline to use the self-hosted Agent for building and deploying the application. 
-	Deploy the containerized application to Azure WebApp.
-	Optionally, use Helm Charts to Deploy the application into the AKS Cluster as a scalable service, for an alternative hosting approach.
